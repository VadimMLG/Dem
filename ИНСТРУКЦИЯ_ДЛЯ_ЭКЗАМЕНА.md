# Универсальная Шпаргалка для Дем-Экзамена 2026

Эта папка содержит **Универсальный Шаблон C# WPF**, который идеально подходит под ЛЮБОЙ вариант 2026 года (Вариант 1, Вариант 3, Вариант 4 и др.).

## План действий на самом экзамене:

### Шаг 1. База Данных (XAMPP / Open Server Panel)
1. Запусти XAMPP (кнопки **Start** напротив Apache и MySQL).
2. Зайди в **phpMyAdmin** (обычно по ссылке `http://localhost/phpmyadmin/`).
3. Создай новую базу данных, назови её, например, `exam2026_db`. Кодировку выбери `utf8mb4_general_ci`.
4. Возьми файлы из выданной тебе папки `import` (например, `Tovar.xlsx`, `user_import.xlsx`).
5. Если это Excel (`.xlsx`), сохрани их как `.csv` (с разделителями запятая или точка с запятой).
6. В phpMyAdmin нажми **Импорт**, загрузи свои `.csv` файлы в соответствующие таблицы (пользователи, товары, заказы, пункты выдачи). Убедись, что структура совпадает с тем, что описано в задании! (Она на 99% будет такой же, как в обувном магазине).

### Шаг 2. Адаптация кода (Подключение к БД)
1. Открой этот проект (`UniversalTemplate.slnx` или `UniversalTemplate.csproj`) в Visual Studio 2026 (или 2022).
2. Открой файл **`db.cs`**.
3. Найди строку: `Database=shoe2026_pu;`
4. Поменяй `shoe2026_pu` на название твоей новой базы данных (`exam2026_db`).
5. Запусти проект (F5). Если данные отображаются — база подключена успешно!

### Шаг 3. Адаптация интерфейса (Визуал под вариант)
1. **Иконка приложения:**
   - В папке с ресурсами на экзамене тебе дадут иконку (`icon.ico` или `icon.png`).
   - Зайди в папку проекта `resources/` и замени там файлы иконок на новые.
2. **Заголовки окон:**
   - Открой `MainWindow.xaml`, `LoginWindow.xaml` и другие окна (двойной клик в Visual Studio).
   - В самом верху кода XAML найди свойство `Title="..."`.
   - Поменяй "Универсальный Магазин" на название магазина из твоего варианта (например, "Парфюмерный магазин").
3. **Фирменный стиль (Цвета):**
   - В задании (в PDF-файле) будет указан фирменный цвет (например, #7FFF00).
   - В XAML файлах или в `App.xaml` (если цвета вынесены туда) найди кнопки (Button) и поменяй их цвет `Background="..."` на нужный по заданию.

### Шаг 4. Сборка (Финальный этап)
1. Убедись, что все кнопки работают (Вход как Гость, Менеджер, Администратор).
2. Проверь добавление товара и фото (фотографии должны лежать в `resources/photos`).
3. Нажми в меню Visual Studio: Сборка -> Собрать решение.
4. Отдай папку `bin/Debug` проверяющим (или как указано в задании).

Удачи! Проект уже написан на оценку "5", тебе осталось только правильно "скормить" ему новые данные.
