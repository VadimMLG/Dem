# ШПАРГАЛКА: Как исправить версию 11:09 на экзамене (Вариант 5)

Если на экзамене у тебя оказалась старая версия (от 11:09) без интернета, тебе нужно будет руками за 5 минут внести эти исправления, чтобы не потерять баллы.

## ШАГ 1: Исправляем цвета строк (файл `UniversalTemplate/Model.cs`)
Открой файл `Model.cs`, найди строку **~50** (свойство `RowBrush`). 
Там написаны неправильные цвета (`#ADD8E6` и `#2E8B57`).

**Как должно стать:**
```csharp
public Brush RowBrush
{
    get
    {
        // 1. Если товара нет на складе -> серый
        if (StockQuantity == 0)
            return new SolidColorBrush((Color)ColorConverter.ConvertFromString("#808080"));
        
        // 2. Если скидка больше 15% -> темно-бирюзовый
        if (DiscountPercent > 15m)
            return new SolidColorBrush((Color)ColorConverter.ConvertFromString("#008080"));
        
        // 3. Иначе -> белый
        return new SolidColorBrush((Color)ColorConverter.ConvertFromString("#FFFFFF"));
    }
}
```

## ШАГ 2: Меняем фильтр с "Поставщика" на "Скидку" (файл `MainWindow.xaml`)
Открой дизайн `MainWindow.xaml`, найди `FilterPanel` (строка ~36).
Удали `TextBlock` "Поставщик:" и `ComboBox` `SupplierComboBox`. 
**Вставь вместо них:**
```xml
<TextBlock Text="Скидка:" VerticalAlignment="Center" Margin="0,0,6,0"/>
<ComboBox x:Name="DiscountComboBox" Width="150" Margin="0,0,16,0" SelectionChanged="DiscountComboBox_SelectionChanged">
    <ComboBoxItem Content="Все диапазоны" IsSelected="True"/>
    <ComboBoxItem Content="0-10,99%"/>
    <ComboBoxItem Content="11-14,99%"/>
    <ComboBoxItem Content="15% и более"/>
</ComboBox>
```

В этом же файле ниже найди `SortComboBox` и **добавь две строки для цены**:
```xml
<ComboBoxItem Content="Цена (по возрастанию)"/>
<ComboBoxItem Content="Цена (по убыванию)"/>
```

## ШАГ 3: Пишем логику фильтра и сортировки (файл `MainWindow.xaml.cs`)
Открой код `MainWindow.xaml.cs`.

**1. Найди и удали:** метод `LoadSuppliersForFilter()` целиком.
**2. В конструкторе `MainWindow` и везде где есть:** удали вызов `LoadSuppliersForFilter();`.

**3. Найди метод `ApplyFiltersAndSorting()`** (строка ~65).
Замени логику фильтрации `supplier` на `discountFilter`:
```csharp
var discountFilter = (DiscountComboBox.SelectedItem as System.Windows.Controls.ComboBoxItem)?.Content?.ToString() ?? "Все диапазоны";
var sort = (SortComboBox.SelectedItem as System.Windows.Controls.ComboBoxItem)?.Content?.ToString() ?? "Без сортировки";

// Логика скидок
if (discountFilter == "0-10,99%")
    query = query.Where(x => x.DiscountPercent >= 0 && x.DiscountPercent < 11);
else if (discountFilter == "11-14,99%")
    query = query.Where(x => x.DiscountPercent >= 11 && x.DiscountPercent < 15);
else if (discountFilter == "15% и более")
    query = query.Where(x => x.DiscountPercent >= 15);
```

Сразу под этим добавь **логику сортировки по цене**:
```csharp
if (sort == "Остаток (по возрастанию)")
    query = query.OrderBy(x => x.StockQuantity);
else if (sort == "Остаток (по убыванию)")
    query = query.OrderByDescending(x => x.StockQuantity);
else if (sort == "Цена (по возрастанию)")
    query = query.OrderBy(x => x.FinalPrice);
else if (sort == "Цена (по убыванию)")
    query = query.OrderByDescending(x => x.FinalPrice);
```

**4. Замени название обработчика:**
Вместо `SupplierComboBox_SelectionChanged` напиши `DiscountComboBox_SelectionChanged`:
```csharp
private void DiscountComboBox_SelectionChanged(object sender, SelectionChangedEventArgs e)
{
    if (!_uiReady) return;
    ApplyFiltersAndSorting();
}
```
